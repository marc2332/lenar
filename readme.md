# 🌑 lenar

**lenar** is a interpreted and toy programming language, written in Rust 🦀.

> ⚠️ I am not an expert in this area.

```rust
println("Running -> " Lenar.version);

let speak = fn(msg) [] {
    println(msg);
    "Hey!"
};

println(speak("Hello?"));

if(isEqual(1 1)) {
    println(speak("Hello again!"));
}
```

- 📦 1 dependency
- 🧩Easy to embed and extend
- 🪶Lightweight
